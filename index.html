Інструкція для GitHub

Щоб використовувати цей пазл на GitHub:

  1. Завантажте цей HTML файл у свій репозиторій
  2. Додайте свої файли (зображення, музика) до репозиторію
  3. Отримайте прямі посилання на ваші файли (використовуючи GitHub Pages або raw.githubusercontent.com)
  4. Введіть URL у відповідні поля вище та натисніть "Застосувати налаштування"
  5. Збережіть та використовуйте оновлений файл
